## 📝 Description
This project is a **Flask-based web application** designed for disaster prediction or related analysis. It uses a machine learning model (placeholder in `model_stub.py`) and a simple front-end built with HTML, CSS, and JavaScript.

## ⚙️ Installation

1. **Clone or extract the repository**  

   git clone <your-repo-url>
   cd minorproject-main

2. **Create a virtual environment (recommended)**  
   python -m venv venv
   source venv/bin/activate   # On Linux/Mac
   venv\Scripts\activate      # On Windows

3. **Install dependencies**  
   pip install -r requirements.txt

## ▶️ Usage
1. **Run the Flask app**  
   python3 app.py

2. **Open your browser** and go to:  
   http://127.0.0.1:8000

## 📁 Key Files
- **app.py** – Launches the Flask server and handles routing.  
- **model_stub.py** – Placeholder for ML model integration.  
- **templates/index.html** – Web interface for user input/output.  
- **static/style.css** – Styles for the app.  
- **static/script.js** – Client-side interactivity.
